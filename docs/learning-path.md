# Blockchain Learning Path

This repository is part of a seven-project interactive blockchain learning series. Each lab focuses on one concept and should grow into a UI-driven visual learning experience.

## Best Learning Path Order

| Order | Lab Repo | Learning Concept |
|---:|---|---|
| 1 | `blockchain-lab-01-chain-basics` | Blocks, hashes, previous hash, validation |
| 2 | `blockchain-lab-02-proof-of-work` | Mining basics, nonce, difficulty |
| 3 | `blockchain-lab-03-pow-simulator` | Interactive PoW difficulty and mining speed visualizer |
| 4 | `blockchain-lab-04-parallel-mining` | Multithreaded mining, performance comparison |
| 5 | `blockchain-lab-05-decentralized-ledger` | Nodes, peer sync, forks, consensus view |
| 6 | `blockchain-lab-06-cryptocurrency-system` | Wallets, balances, transactions, mempool |
| 7 | `blockchain-lab-07-smart-contracts` | Contract state, function calls, gas-style execution |
